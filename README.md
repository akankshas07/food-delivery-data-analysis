# Food Delivery Data Analysis

This repository contains a Jupyter Notebook demonstrating end-to-end data integration and analysis for a food delivery system.

## Project Overview
The project integrates datasets from multiple formats:
- CSV file containing order transaction data
- JSON file containing user details
- SQL database containing restaurant information

The datasets are merged using Pandas (LEFT JOIN) and analyzed to answer business and analytical questions.

## Tools & Technologies
- Python
- Pandas
- Google Colab
- SQL (SQLite)
- GitHub

## Files in This Repository
- `food_delivery_analysis.ipynb` – Main Jupyter Notebook
- `README.md` – Project documentation

## Key Analysis Performed
- Revenue analysis by city and cuisine
- Membership-based order comparison (Gold vs Regular)
- Average order value calculation
- Rating-based order insights
- Quarterly revenue trends
- User behavior analysis

## Purpose
This project was completed as part of a data analytics internship / hackathon assessment.
